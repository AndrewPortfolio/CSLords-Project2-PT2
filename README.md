# CSLords-Project2-PT2

## NN Classifier and a Validator 
* NN Classifier: Is trained on a dataset, calculate k for data points then return the label prediction for said data points 
* Validator: Uses the Classifier, dataset, a feature subset, and returns the accuracy of the classifier on the given dataset

## NN Classifier 
* Train: Passes in data points and then keeps training instances in memory 
* Test: input is a test point and return its predicted label
* Euclidean: Calc the distance between test and trained points
